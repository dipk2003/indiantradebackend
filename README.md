# 🏪 Indian Trade Market — MVP Backend

The Spring Boot backend for Indian Trade Market (ITM), a B2B marketplace platform. Provides REST APIs for user management, product catalog, and trade operations. MVP phase complete.

![Spring Boot](https://img.shields.io/badge/Spring_Boot-Backend-6DB33F?logo=springboot&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-Containerized-2496ED?logo=docker&logoColor=white)

---

## ✨ Features

- **REST APIs** — Full CRUD for users, products, and trades
- **Authentication** — JWT-based auth with Spring Security
- **Email Integration** — SMTP email service
- **API Documentation** — Comprehensive API docs included
- **Docker Support** — Multi-container with Docker Compose
- **Multi-Platform Deploy** — Render, AWS, and Docker deployment scripts
- **3 Releases** — Production-ready with published releases

## 🚀 Quick Start

```bash
git clone https://github.com/dipk2003/indiantradebackend.git
cd indiantradebackend

# With Docker
docker-compose up --build

# Or with Maven
mvn clean install
mvn spring-boot:run
```
